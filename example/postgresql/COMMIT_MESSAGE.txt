Add PostgreSQL support and example for SureSQL abstraction

This commit adds full PostgreSQL database support to SureSQL's abstraction
layer and includes a comprehensive example demonstrating the integration.

## Core Changes

### SureSQL Package (suresql.go)
- Enhanced NewDatabase() to support multiple DBMS types via configuration
- Added newPostgreSQLDatabase() function for PostgreSQL connections
- Added newRQLiteDatabase() function (refactored from existing code)
- Implemented automatic driver selection based on DBMS_TYPE configuration
- Supports: POSTGRESQL, POSTGRES, RQLITE

### Configuration (config.go)
- Added DBMS field to SureSQLDBMSConfig struct
- Added DBMS_TYPE environment variable support (defaults to RQLITE)
- Maintains backward compatibility with existing RQLite configurations

### Features Demonstrated
